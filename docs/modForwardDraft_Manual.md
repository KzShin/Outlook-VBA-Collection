# Outlook Forward Draft Creator (modForwardDraft)
これは、OutlookのVBAマクロです。選択したメールを指定した宛先への「転送メール」として自動生成し、下書きフォルダに保存します。
よく転送する宛先を事前に設定ファイルに登録しておくことで、ワンクリックで宛先・定型文が入力済みの転送用下書きを作成できます。複数の宛先が登録されている場合は、専用の選択ウィンドウ（ダイアログ）が表示されます。
## 特徴
* **ワンクリック下書き作成**: 選択したメールの転送アイテムを瞬時に生成し、下書き保存します。
* **テキスト形式への強制変換**: 転送元のメールがHTML形式やリッチテキスト形式であっても、自動的にプレーンテキスト形式（テキストメール）に変換して転送します。
* **定型文の自動挿入**: 本文の先頭に「宛先名」と「転送します。」という定型メッセージを自動的に挿入します。
* **宛先のリスト管理**: 外部ファイル（`config.ini`）で複数の転送先を管理できます。
* **専用選択UI**: 複数の転送先が設定されている場合、見やすいリストボックス付きのウィンドウ（ユーザーフォーム）を表示して選択させます。
* **共通ログ機能**: 共通モジュール `modLogger` を使用し、実行ログを記録します。
## 必要要件
* Windows 10 / 11
* Microsoft Outlook (Classic Desktop)
* **共通モジュール**: `modLogger.bas` (本リポジトリに含まれる共通ログモジュール)
* **専用ユーザーフォーム**: `frmSelectDest.frm` / `frmSelectDest.frx`
## インストール
1. このリポジトリのファイルをダウンロードします。
2. Outlookを起動し、`Alt + F11` でVBAエディタを開きます。
3. **モジュールとフォームのインポート**:
    * `ファイル` > `ファイルのインポート` から以下のファイルをインポートします。
        1. `src/modLogger.bas` (共通ログモジュール)
        2. `src/modForwardDraft.bas` (メイン機能モジュール)
        3. `src/frmSelectDest.frm` (宛先選択用ユーザーフォーム) ※ `.frx` ファイルは自動的に読み込まれます。
4. **ThisOutlookSessionの設定**:
    * VBAエディタ左側の `Project1` > `Microsoft Outlook Objects` > `ThisOutlookSession` をダブルクリックします。
    * 以下のプロシージャを追加します（リボンやクイックアクセスツールバーからの呼び出し用です）。
        ```vb
        ' 選択したメールを下書きへ転送するマクロ
        Public Sub 転送メールを下書き作成()
            modForwardDraft.CreateForwardDraft
        End Sub
        ```
## 設定 (Configuration)
転送先のアドレスと表示名は、共通設定ファイル `%APPDATA%\OutlookVBA\config.ini` の **`[ForwardMail]`** セクションで設定します。
1. **設定ファイルの準備**:
    * エクスプローラーで `%APPDATA%\OutlookVBA\` を開きます。
    * `config.ini` をテキストエディタで開きます。
2. **設定の編集**:
    * 以下のセクションを追記・編集します。
    * 書式は `任意のキー名 = ウィンドウに表示する名前 , メールアドレス` です（カンマ区切り）。
    * **重要**: 保存時の文字コードは必ず **UTF-8 (BOMなし)** にしてください。
        ```ini
        [ForwardMail]
        # 書式: 任意のキー = 転送先の表示名 , メールアドレス
        # （複数設定すると、マクロ実行時に選択ウィンドウが表示されます。1つだけの場合は自動選択されます。）
        Dest1=営業部 山田, yamada@example.com
        Dest2=サポート窓口, support@example.com
        Dest3=プロジェクトチーム全体, team-all@example.com
        ```
## 使い方
1. Outlookのメール一覧から、転送したいメールを選択します。
2. マクロ `転送メールを下書き作成` を実行します。（リボンやクイックアクセスツールバーに登録しておくと便利です）
3. **（宛先が複数ある場合）** 「転送先の選択」ウィンドウが表示されるので、リストから宛先を選んで「選択」ボタンをクリックします。
4. 処理が完了するとメッセージが表示され、Outlookの「下書き」フォルダにテキスト形式に変換された転送メールが保存されます。
## ログ
本ツールは `modLogger` を使用して動作ログを記録します。
* **ログ保存場所**: `%APPDATA%\OutlookVBA\logs\`
* **ファイル名**: `yyyy-mm-dd.log` (日付ごとのローテーション)
* **ログ出力例:**
    ```text
    2026/03/03 10:00:00.123 [260303-100000-FWD] [ForwardDraft] === START 転送メール作成処理 ===
    2026/03/03 10:00:00.150 [260303-100000-FWD] [ForwardDraft] 対象メール: 【重要】システムメンテナンスのお知らせ
    2026/03/03 10:00:00.200 [260303-100000-FWD] [ForwardDraft] 複数設定あり、選択画面を表示
    2026/03/03 10:00:05.500 [260303-100000-FWD] [ForwardDraft] ユーザー選択: 営業部 山田 <yamada@example.com>
    2026/03/03 10:00:05.800 [260303-100000-FWD] [ForwardDraft] 下書きへ保存完了: FW: 【重要】システムメンテナンスのお知らせ (テキスト形式変換済)
    2026/03/03 10:00:05.810 [260303-100000-FWD] [ForwardDraft] === END 転送メール作成処理 ===
    ```
