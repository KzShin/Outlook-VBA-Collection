# Outlook Mail Open Controller (modMailOpen)
これは、OutlookのVBAマクロです。メール一覧でのダブルクリックやEnterキーによる誤ったポップアップ開封を防ぎ、**特定のショートカットキーを押したときのみメールを開く**ように制御します。
メールを次々と選択して閲覧する際、誤爆でウィンドウが大量に開いてしまうストレスを解消します。
## 特徴
* **誤爆防止 (開封ブロック)**: 設定をオンにすると、通常のマウスクリックやEnterキーでのメール開封を完全にブロックし、無反応にします。
* **ショートカットでのみ開封**: クイックアクセスツールバー（QAT）と連携することで、`Alt + 1` などのショートカットキー操作時のみメールをポップアップ表示します。
* **スマートな例外処理**: 新規作成メール、返信メール、下書きなど、まだ送信されていないアイテムはブロックの対象外となり、通常通り開くことができます。
* **設定によるオン・オフ機能**: 機能の有効・無効を外部の設定ファイルから簡単に切り替えられます（デフォルトは無効）。
* **共通ログ機能**: 共通モジュール `modLogger` を使用し、実行ログを記録します。
## 必要要件
* Windows 10 / 11
* Microsoft Outlook (Classic Desktop)
* **共通モジュール**: `modLogger.bas` (本リポジトリに含まれる共通ログモジュール)
## インストール
1. このリポジトリのファイルをダウンロードします。
2. Outlookを起動し、`Alt + F11` でVBAエディタを開きます。
3. **モジュールのインポート**:
    * `File` > `Import File` から以下の2つのファイルをインポートします。
        1. `src/modLogger.bas` (共通ログモジュール)
        2. `src/modMailOpen.bas` (メイン機能モジュール)
4. **ThisOutlookSessionの設定**:
    * VBAエディタ左側の `Project1` > `Microsoft Outlook Objects` > `ThisOutlookSession` をダブルクリックします。
    * 同梱の `src/ThisOutlookSession.cls` の中身をすべてコピーし、貼り付けて上書きします。
    * （※本機能はイベントフックを利用するため、`Application_Startup`、`myExplorer_SelectionChange`、`myItem_Open` へのコード追加が必須です）
## 設定 (Configuration)
本機能は**デフォルトではオフ（False）**に設定されています。利用を開始するには、共通設定ファイル `%APPDATA%\OutlookVBA\config.ini` の **`[MailOpen]`** セクションを編集して有効化します。
1. **設定ファイルの準備**:
    * エクスプローラーで `%APPDATA%\OutlookVBA\` を開きます。
    * `config.ini` をテキストエディタで開きます（ファイルがない場合は `configs/config.sample.ini` をコピーして作成）。
2. **設定の編集**:
    * 以下のセクションを追記・編集します。
    * **重要**: 保存時の文字コードは必ず **UTF-8 (BOMなし推奨)** にしてください。
        ```ini
        [MailOpen]
        # [メール開封制御] ショートカットキーでのみメールを開く機能を有効にするか (True / False)
        # Trueにすると、通常のマウスクリック等ではメールが開かなくなります。
        EnableShortcutOpen=True
        ```
## 使い方（ショートカットキーの割り当て）
本機能は、Outlookの「クイックアクセスツールバー (QAT)」を利用してショートカットキーを割り当てます。
1. **ボタンの登録**:
    * Outlookのメイン画面左上（またはリボン下）にあるクイックアクセスツールバーの右端の「▼」をクリックし、「その他のコマンド」を開きます。
    * 「コマンドの選択」のプルダウンを「マクロ」に変更します。
    * `Project1.modMailOpen.OpenSelectedMail` を選択し、「追加」ボタンで右側のリストへ移動させます。
2. **ショートカットキーの決定**:
    * 右側のリストに追加したマクロを選択し、リスト右横にある「上へ」ボタン（▲）を使って**一番上（1番目）**に移動させます。
    * 「OK」で設定画面を閉じます。
3. **実行**:
    * QATの1番目に登録されたコマンドは、自動的に **`Alt` + `1`** がショートカットキーとして割り当てられます。
    * 一覧からメールを選択し、`Alt + 1` キーを押すとメールがポップアップで開きます。マウスのダブルクリックでは開かなくなります。
## ログ
本ツールは `modLogger` を使用して動作ログを記録します。
* **ログ保存場所**: `%APPDATA%\OutlookVBA\logs\`
* **ファイル名**: `yyyy-mm-dd.log` (日付ごとのローテーション)
* **ログ出力例:**
    ```text
    2026/02/16 13:00:00.123 [260216-130000-OPEN] [MailOpen] === START 選択メール開封処理 ===
    2026/02/16 13:00:00.130 [260216-130000-OPEN] [MailOpen] 対象メールを開封します: 〇〇プロジェクト進捗について
    2026/02/16 13:00:00.140 [260216-130000-OPEN] [MailOpen] === END 選択メール開封処理 ===
    ```
